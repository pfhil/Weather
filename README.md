В папке WeatherWinFormsUI создать файл App.config со следующим содержимым:

<?xml version="1.0" encoding="utf-8" ?>
<configuration>
  <appSettings>
    <add key="AppId" value="{YourAppIdForOpenweathermap}"/>
  </appSettings>
</configuration>

Где {YourAppIdForOpenweathermap} поставить свой ключ
